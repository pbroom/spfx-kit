# Hello Card

Built-in SPFx Kit example: an SPFx 1.21.1 web part with a lab adapter at
`.spfx-kit/lab/register.tsx`. It appears in the lab automatically after
`npm run sync:lab` so a fresh clone has something to preview.

This committed copy is read-only reference material. To build, edit, or
export it as your own app, copy it into the ignored managed area first (see
`examples/README.md`).
